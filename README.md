# Конвертер величин (Вариант 2)

**ФИО студента:** Сиразиев Алмаз Ильнарович 
**Вариант:** 2  
**Дисциплина:** МДК.01.02 Поддержка и тестирование программных модулей  

## Описание
Графическое WPF-приложение для конвертации величин (метры - километры, °C - °F). Реализована валидация ввода, обработка исключений и полное покрытие логики автоматизированными тестами (MSTest).
